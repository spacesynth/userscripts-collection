# userscripts-collection
A collection of custom free and public userscripts.  

## Removal of small thumbnail script
Please use this user script for small thumbnails:  
https://greasyfork.org/en/scripts/405614-youtube-polymer-engine-fixes  

## Removal of Youtube Dark Mode + No Autoplay
![alt text[]()](https://github.com/spacesynth/userscripts-collection/blob/master/utility/uBO_settings.png?raw=true)  
Do it simpler in uBlock Origin and the "My Filters" pane and the above and below set.  
```youtube.com##+js(trusted-set-cookie-reload, PREF, tz=Europe.Berlin&f5=30000&f6=400&gl=US&f7=100)```  

## Removal of X dim theme script
Use this instead with the Firefox Stylus addon:  
```x.com##+js(trusted-set-cookie-reload, night_mode, 1)```  
And this:  
https://userstyles.world/style/26534/twitter-dark-to-dim  
Or explore here:  
https://uso.kkx.one/browse/styles
